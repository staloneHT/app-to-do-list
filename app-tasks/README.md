Aplicación de Lista de Tareas (To-Do App)

Aplicación web desarrollada con React que permite gestionar tareas de forma simple e intuitiva. El usuario puede agregar nuevas tareas, marcarlas como completadas y eliminarlas.

Este proyecto fue creado como práctica para fortalecer conocimientos en React, especialmente en el manejo del estado, uso de hooks y comunicación entre componentes.



Funcionalidades

Agregar nuevas tareas

Marcar tareas como completadas

Eliminar tareas

Generación de identificadores únicos para cada tarea

Interfaz dinámica sin recarga de página



Tecnologías utilizadas

React

JavaScript (ES6+)

HTML5

CSS3

React Hooks (useState)

UUID

React Icons



Conceptos aplicados

Componentes funcionales

Props

Manejo de estado con useState

Renderizado condicional

Manejo de eventos

Inmutabilidad del estado




Ejecución del proyecto


Clonar el repositorio:

git clone https://github.com/tu-usuario/tu-repositorio.git


Instalar dependencias:

npm install


Ejecutar la aplicación:

npm start
